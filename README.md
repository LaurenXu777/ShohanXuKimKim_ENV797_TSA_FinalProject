# TSA-Team-Project
Forecasting U.S. gas prices following geopolitical shocks (Time Series Project)

## Overview
This project aims to forecast U.S. gas prices following geopolitical shocks 
(e.g., Russia-Ukraine war, Venezuela, Iran).

## Data
- Source:
  - EIA Weekly Gas Prices
  - Crude Oil Prices (WTI/Brent)
  - Gasoline Stocks
- Raw data: Raw_Data/
- Processed data: Processed_Data/

## Methodology
- Exploratory Data Analysis (EDA)
- Data preprocessing and cleaning
- Stationarity tests (ADF)
- Time series modeling:
  - ARIMA/SARIMA
  - ARIMAX
  - ETS
  - Prophet
- Evaluation: RMSE, MAE

## Structure
TSA-Team-Project/
